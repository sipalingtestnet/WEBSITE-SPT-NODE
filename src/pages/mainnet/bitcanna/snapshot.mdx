## Intruction

### install lz4
```
sudo apt install lz4 -y
```

### reset chain data
```
sudo systemctl stop bcnad
bcnad tendermint unsafe-reset-all --home $HOME/.bcna --keep-addr-book
```

### download data

```
curl -o - -L https://snapshots.bitcanna.spt-node.my.id/bitcanna/bitcanna-snapshot-20230205.tar.lz4 | lz4 -c -d - | tar -x -C $HOME/.mars
```

### start node

```
sudo systemctl restart bcnad
sudo journalctl -fu bcnad -o cat
```
